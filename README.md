## BitMoi - 매매 주문 (OrderBook)
***

## 기본 설명
> 코인 매수/매도 주문 및 주문 취소를 진행한다.

## 동작 방식

[//]: # (```)
> 매수/매도 주문
  >* 지갑에 있는 KRW 보다 주문 금액이 적을 때 코인 매수/매도 주문
  >* 매매 내용에 대한 정보 카프카로 전송

> 매수/매도 거절
  >* 지갑에 있는 KRW 보다 주문 금액이 많을 때 코인 매수/매도 거절

> 매수/매도 취소
  >* 매매를 취소하고자 하는 매매 건을 선택해 코인 매수/매도 취소

> 매수/매도 목록
  >* 매수/매도를 진행한 목록에 대해 불러옴
